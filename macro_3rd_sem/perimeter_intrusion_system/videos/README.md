# Videos Directory

Place your video files here for processing by the Perimeter Intrusion Detection System.

## Supported Formats
- MP4
- AVI
- MOV
- MKV
- WMV

## Usage
```bash
python main.py --video videos/your_video.mp4
```

## Sample Videos
You can use any video that contains people walking or moving objects. Some good sources for test videos:
- Security camera footage
- Crowd surveillance videos
- Public space recordings
- Traffic monitoring videos

## Tips for Better Detection
- Use videos with clear, well-lit scenes
- Ensure people are clearly visible and not too far from camera
- Avoid heavily crowded scenes for initial testing
- Test with videos where people enter/exit the frame naturally
